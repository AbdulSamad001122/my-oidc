Auth Code Flow

Standard OAuth 2.0 Authorization Code grant. Issue and exchange codes securely for access and ID tokens via your dedicated token endpoint.

🗄️

MongoDB Architecture

Built on a solid Mongoose schema. Persistent tracking of Users, Client Applications, and ephemeral Auth Codes for extreme reliability.

🔑

RS256 JWT Signing

Secure ID tokens generated with `node-jose` using RSA-SHA256 signatures. Automatically exposed via your `/.well-known/jwks.json` endpoint.

👥

Multi-Account Sessions

Like Google, users can maintain multiple active sessions. The built-in account chooser lets them seamlessly switch identities during authorization.

📡

OIDC Discovery

Plug and play with any library. Your provider implements `.well-known/openid-configuration` so client libraries can autodiscover endpoints.

Developer Portal

A ready-to-use admin dashboard for developers to register applications, manage redirect URIs, and instantly copy their Client IDs and Secrets.

Drop it into your app in seconds.

Because it fully implements the OpenID Connect spec, you can use standard libraries across React, Next.js, Go, or Python.

  • 1
    Register your app in the Developer Portal
  • 2
    Copy your client_id
  • 3
    Redirect users to /o/authorize
app.js
const url = \`\${OIDC_URL}/o/authorize\`;
const params = new URLSearchParams({
  client_id: 'YOUR_CLIENT_ID',
  redirect_uri: 'https://app.com/callback',
  response_type: 'code',
  state: 'random_state_string'
});

// Send user to MyOIDC Login / Account Chooser
window.location.href = \`\${url}?\${params}\`;
